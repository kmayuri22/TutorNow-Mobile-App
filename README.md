# TutorNow Mobile App

This repository contains the native Android application for the **TutorNow** platform. The app is written in Kotlin and built using Jetpack components, Material Design 3, and Retrofit to communicate with the FastAPI backend.

## 📱 Features

- **Student Role**: Tutor search, filtering, detailed profiles, booking calendar (Online/In-Person), and booking history tracking.
- **Tutor Role**: Availability manager, incoming requests dashboard, accept/reject workflows, and session schedule.
- **Admin Role**: Platform metrics dashboard and user audit management.

## 📁 Project Structure

The Android Studio project is located inside the [`TutorNow/`](./TutorNow) subdirectory.
